# go-server
A basic Go server project


- Start your terminal
- go to the folder where all files are stored
- run go build on terminal
- run go run main.go
- go to browser
- search http://localhost:6969/ to get static website
- search http://localhost:6969/hello to get hello
- search http://localhost:6969/form.html to get a form format to enter your name and address


 #Note: Make sure you have all required libraries installed
